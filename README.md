# streaming
Generate html pages for streaming links

Modify Search_name with one of the options below to generate the html file 

options : nr, nr2, n3, nr4, nr5, kids, horror, doc, alien, anim, comedies, disaster, fight, gangster, or ghost

if initial = True then Search_name first letter is used !

options : 0, a, e, i, m, q, t, u 

NEW :  the command line below may be used w/wo -n, -i or -d 

$ python3 makehtmlfile.py [-n name] [-i] [-d]     (name is one of the options above, -i is for initial search, -d debug verbose)
